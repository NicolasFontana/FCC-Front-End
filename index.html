SOBRE MI

Quién soy?

Soy una persona autodidacta y persistente, disfruto el proceso de aprendizaje al estar enfrentándome a un nuevo desafío. Estudiar ingeniería en éstos últimos años me ha brindado herramientas para la resolución de problemas, logré abordarlos aplicando lógica y razonamiento para llegar a un óptimo desenlace.

Me apasiona el desarrollo de la tecnología, programación e innovación del mundo digital. Estoy buscando desarrollarme en el ámbito profesional como Front End Developer con proyectos que permitan desplegar mis habilidades. Si bien se trata de un camino que está en constante evolución, me entusiasma explorar nuevas tecnologías y mantenerme actualizado.

PROYECTOS

Pomodoro Timer

HTML Bootstrap React

Este proyecto consiste en un temporizador basado en la técnica Pomodoro la cual sirve para administrar el tiempo en bloques de trabajo y descanso, generalmente 25 y 5 minutos respectivamente. Estos mismos se pueden modificar a gusto de cada uno. La técnica Pomodoro consiste en usar un temporizador para dividir el trabajo en bloques de tiempo enfocados (generalmente 25 minutos) separados por un breve descanso, generalmente de cinco minutos

Drum Machine

HTML CSS Bootstrap React

Un proyecto divertido que consistía en armar una máquina de sonidos, cada tecla reproduce un sonido distinto el cual se va almacenando y luego se puede reproducir toda la secuencia, pudiendose modificar el volumen y velocidad con el que se reproduce. Además se puede borrar la última nota de la secuencia para poder encontrar tu ritmo!
